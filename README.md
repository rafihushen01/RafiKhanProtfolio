# Rafi Khan Portfolio Website

This repository contains the source code of my official portfolio website.

The site is built with a custom React + TypeScript + Vite setup, GSAP motion, and Three.js scenes.  
It has been fully personalized with my own profile, projects, contacts, assets, and green signature visual theme.

- Contact + Social Links

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```

## Important Note About GSAP Plugins

This project uses the standard `gsap` package (including bonus plugins that are now free).  
Do not use `gsap-trial` in production deployments.

Official docs:

https://gsap.com/docs/
